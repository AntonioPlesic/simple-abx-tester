Encoders & decoders

Encoders

App can use following encoders in automated test modes to encode a lossless file into lossy

  • LAME v3.99.5 - for encoding into mp3

Decoders

If user doesn't provide own decoded files, supported formats will be decoded in-app using these decoders

  • mpg123 (libmpg) that comes with LAME v3.99.5 (may differ from other mpg123 versions) - for decoding mp3
  • libflac v1.2.1 - for decoding flac files

More encoders/decoders/formats are planned
