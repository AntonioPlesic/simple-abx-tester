Simple ABX Tester

Simple ABX Tester is a simple and intuitive audio abx application that enables you to perform double blind listening tests of various audio formats on your Android device.

Testing impact of mp3 bitrates on sound quality

For testing mp3, app provides automatic test preparation mode: all you need to do is select a lossless source file and how you wish to encode it and app will do the rest:

  1. decode the lossless source to raw pcm for playback
  2. encode the source to mp3 of the desired bitrate/quality, using LAME v3.99.5 encoder
  3. decode the encoded file to raw pcm for playback, using LAME supplied version of mpeg123/limpeg decoder
  4. synchronize the two raw files, so there are no audible clicks or silences while switching between tracks

Self-prepared mode

For testing all other file types or encoders. You can easily test other file types and encoders/decoders by providing the decoded (to wav) files yourself. Explained in detail here.

Intuitive interfaces

App provides two types of switching interfaces:
  • ABXY - a Foobar2000 ABX plugin inspired ABXY interface, where all you need to do is match any two tracks you think sound the same using drag and drop.

  • ABX - you are provided a reference track, and you must answer which of the two choice tracks sounds the same as reference. Alternatively, if you prefer it that way, you can answer by choosing the choice track that is different from the reference.
In both interfaces, switching between the tracks is almost instantaneous, and there are no clicks or pauses while the switching takes place (provided the files are synchronized), so there are no tells you could use to cheat the test.

Simple statistics

After each answer, app tells you whether you answered correctly and counts how many times you were correct. More importantly, it also tells you the likelihood that you were guessing, i.e. a chance of achieving the same or better result by simply guessing.
